OccupationalDataSection

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/odh-OccupationalDataSection-model

Section in the OccupationalDataForHealth composition, containing the person’s occupational data, past and present.

This logical model was published on Mon Apr 08 00:00:00 EDT 2019 as a draft by HL7.

This IG does not define a corresponding profile for this logical model.
