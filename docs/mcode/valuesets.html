Primary local value sets used in this Implementation Guide

Name Definition
ActiveOrInactiveVS

Status values indicating the resource is either active or inactive.
BloodPressureBodyLocationVS

Body locations for blood pressure measurement
BloodPressureCuffSizeVS

Sizes of blood pressure cuffs.
BloodPressureMethodVS

Method used to determine blood pressure.
BodyHeightLengthMethodVS

Method used to determine body height or length.
BodyHeightPreconditionVS

Circumstance for measuring body height.
BodyLocationVS

Codes that describe normal and pathologic anatomic systems, regions, cavities, and spaces.
BodyPositionVS

Human body positions associated with actions or observations.
BodyWeightMethodVS

Method used to determine body weight.
BodyWeightPreconditionVS

Circumstances for body weight measurement. The choices are limited by availability of SNOMED terms, and could contain terms like pre- and post-dialysis, mother ante- and post-delivery, etc.
CancerBodyLocationVS

Codes describing the location of primary or secondary cancer. The value set includes all codes from ICD-O-3 topology and SNOMED-CT body locations.
CancerDiseaseStatusEvidenceTypeVS

The type of evidence backing up the clinical determination of cancer progression.
CancerDisorderVS

A broad cancer-related value set containing both primary and secondary tumor types, with codes from ICD-10, ICD-O-3, and SNOMED CT, including both diagnosis and histology/morphology/behavior codes.
CancerHistologicGradeVS

The classification for the grade of the tumor. This is a subset of the LOINC answer list LL213-0, which represents allowable values for NAACCR data item #440. The original answer list is outdated, as it includes terms pointing to the lineage of hematopoietic/lymphoid neoplasms, which have been retired by NAACCR as of version 18 Data Standards and Data Dictionary.
CancerStageTimingVS

When staging was done, relative to treatment events (qualifier).
CancerStagingSystemVS

The system used to identify cancer/tumor stage
CardiopulmonaryPreconditionVS

Circumstances for cardiopulmonary measurements.
ComorbidConditionCodeVS

Preferred SNOMED and ICD-10 codes for common comorbid conditions. Aligns with Elixhauser comorbidity scale. ICD-10 codes are drawn from https://www.hcup-us.ahrq.gov/toolssoftware/comorbidityicd10/comformat_icd10cm_2019_1.txt
ConditionCategoryVS

A category assigned to the condition, for example, a disease, concern, symptom, functional impairment, or structural abnormality. Based on http://fhir.org/guides/argonaut/ValueSet/condition-category but with SNOMED CT codes.
ConditionStatusTrendVS

Standardized value set that describes the trend or status of a disease, condition, or ability. Based on LL2751-7, LL4721-8.
DetectedNotDetectedVS

Value set consisting of the answers Detected (Present) and NotDetected (Absent). The terms detected and not detected are preferred because they reflect outcomes of experimental observation, while present and absent suggest ground truth. This value set reconciles or combines LOINC answer lists LL956-4, LL750-1, LL744-4, LL955-6, LL3950-4, LL3250-9, LL2872-1, LL2410-0, LL2355-7.
GeneticTestMethodVS

The method used to perform the genetic test. Examples include Fluorescent in situ hybridization (FISH), polymerase chain reaction-based assays (PCR), Sanger sequencing, and next-generation sequencing (NGS). Contains the following values from NCI Thesaurus
GenomicSourceClassVS

The genomic class of the specimen being analyzed: Germline for inherited genome, somatic for cancer genome, and prenatal for fetal genome. (source: LOINC). These are answers to the question specimen source (SCT#703065002).
HistologyMorphologyBehaviorVS

The histologic type codes can be taken either from ICD-O-3 morphology codes (including behavior suffix) or SNOMED CT neoplasm morphology codes. For primary cancers, the ICD-O-3 behavior suffix should be /1, /2, or /3. For secondary cancers, the ICD-O-3 code should be /6 or /9.
PrimaryCancerDisorderVS

Types of malignant neoplastic disease. The value set includes SNOMED CT disorder hierarchy codes from SCT#363346000 ‘Malignant neoplastic disease (disorder)’ but excludes those descending from SCT#128462008 ‘Secondary malignant neoplastic disease (disorder)’. The value includes ICD-10-CM codes for malignant neoplasms and neoplasms of uncertain or unspecified behavior. For ICD-10-CM coding of the primary disease, the disorder code must be SCT#363346000 (Malignant neoplastic disease), and histology/morphology/behavior code and the topology code (body site) and should be specified separately in the HistologyMorphologyBehavior extension and BodySite, respectively.
QuantitativeResultInterpretationVS

Interpretations of abnormal quantitative results.
RadiationProcedureVS

Radiation oncology AND/OR radiotherapy (procedure)
SecondaryCancerConditionVS

Types of secondary malignant neoplastic disease. Includes SNOMED CT disorder hierarchy codes for secondard malignant neoplastic disease, as well as ICD-10-CM codes for malignant neoplasms and neoplasms of uncertain or unspecified behavior. If body site is not precoordinated (implied by the code), it shall be separately specified. For ICD-10-CM coding of the secondary disease, the disorder code should always be SCT#128462008 (Secondary malignant neoplastic disease), and histology/morphology/behavior code and the topology code (body site) and should be specified separately in the HistologyMorphologyBehavior extension and BodySite, respectively.
StageGroupVS

Classification of the extent of the disease, according to the classification system’s rules, typically taking into account several anatomic and prognostic elements. This value set intends to cover content contained in LOINC answer lists LL240-3 and LL234-6.
SurgicalProcedureVS

Surgical procedure (procedure)
TNMPrimaryTumorClassificationVS

Possible classifications of the primary tumor according to TNM staging rules.
TNMTumorStagingSystemVS

System used for TNM classification. Includes the following codes: SCT#444256004 ‘American Joint Commission on Cancer, Cancer Staging Manual, 6th edition neoplasm staging system (tumor staging)’ SCT#443830009 ‘American Joint Commission on Cancer, Cancer Staging Manual, 7th edition neoplasm staging system (tumor staging’ #AJCC8 ‘American Joint Commission on Cancer, Cancer Staging Manual, 8th edition neoplasm staging system (tumor staging)’ SCT#258235000 ‘International Union Against Cancer (tumor staging)’
TreatmentIntentVS

The purpose of a treatment. A simplified set taken from SNOMED, only including curative and palliative.
TreatmentTerminationReasonVS

Values used to describe the reasons for stopping a treatment. Includes code for ‘treatment completed’ as well as codes for unplanned (early) stoppage. Applies to medications and other treatments that take place over a period of time, such as radiation treatments.
TumorMarkerTestVS

A tumor marker is a substance found in tissue or blood or other body fluids that may be a sign of cancer or certain benign (noncancer) conditions. This value set of LOINC codes will include a comprehensive list of tumor markers across all cancers, and regardless of method or specimen type. Subsets specific to a cancer type will be created as needed. Sources vary but include https://labtestsonline.org/tests/tumor-markers. At the current time, profiles for the LOINC tests do not exist.
