Composition

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-Composition-model

A composition represents a set of Entries sharing common provenance, a document such as an extract from a patient record, a knowledge artifact, or a catalog definition.

This logical model was published on Tue May 07 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
