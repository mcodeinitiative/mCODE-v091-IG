Primary local value sets used in this Implementation Guide

Name Definition
ActiveOrInactiveVS

Status values indicating the resource is either active or inactive.
BloodPressureBodyLocationVS

Body locations for blood pressure measurement
BloodPressureCuffSizeVS

Sizes of blood pressure cuffs.
BloodPressureMethodVS

Method used to determine blood pressure.
BodyHeightLengthMethodVS

Method used to determine body height or length.
BodyHeightPreconditionVS

Circumstance for measuring body height.
BodyLocationVS

Codes that describe normal and pathologic anatomic systems, regions, cavities, and spaces.
BodyPositionVS

Human body positions associated with actions or observations.
BodyWeightMethodVS

Method used to determine body weight.
BodyWeightPreconditionVS

Circumstances for body weight measurement. Some of these terms are unavilable in SNOMED
CancerDiseaseStatusEvidenceTypeVS

The type of evidence backing up the clinical determination of cancer progression.
CancerDiseaseStatusVS

Concepts describing the status of disease in cancer. Analogous values to LOINC list LL4721-8.
CancerDisorderVS

Types of malignant neoplastic disease. Includes SNOMED CT disorder hierarchy codes for malignant neoplastic disease, as well as ICD-10-CM codes for malignant neoplasms and neoplasms of uncertain or unspecified behavior. Both SNOMED CT and ICD-10-CM can precoordinate the cancer location and the histologic type, which will result in some overlap between Cancer Condition and Cancer Histologic Type. For ICD-10-CM in particular, if the histology is known it must be coded.
CancerHistologicGradeVS

The classification for the grade of the tumor. This is a subset of the LOINC answer list LL213-0, which represents allowable values for NAACCR data item #440. The original answer list is outdated, as it includes terms pointing to the lineage of hematopoietic/lymphoid neoplasms, which have been retired by NAACCR as of version 18 Data Standards and Data Dictionary.
CancerHistologicTypeVS

Neoplasm (morphologic abnormality)
CancerStageTimingVS

When staging was done, relative to treatment events (qualifier).
CancerStagingSystemVS

The system used to identify cancer/tumor stage
CardiopulmonaryPreconditionVS

Circumstances for cardiopulmonary measurements.
ComorbidConditionCodeVS

Preferred codes for common comorbid conditions. Aligns with Adult Comorbitity Evaluation (ACE-27).
ConditionCategoryVS

A category assigned to the condition, for example, a disease, concern, symptom, adverse reaction, functional impairment, or structural abnormality.
Equivalence-g-per-L-VS
GeneticTestMethodVS

The method used to perform the genetic test. Examples include Fluorescent in situ hybridization (FISH), polymerase chain reaction-based assays (PCR), Sanger sequencing, and next-generation sequencing (NGS).
GenomicSourceClassVS

The genomic class of the specimen being analyzed: Germline for inherited genome, somatic for cancer genome, and prenatal for fetal genome. (source: LOINC).
MetastasesVS

Code for secondary malignant neoplastic disease
MorphologyBehaviorVS

The morphologic behavior of the cancer. These are the suffix to the ICD-O-3 histologic type codes. Codes are presented as local codes because they are portions of the full code to be post-coordinated, and not full ICD-O-3 codes.
PositiveNegativeVS

Interpretation of a test result as positive, negative or equivocal. Equivalent values to LOINC LL2009-0.
QuantitativeLaboratoryCodes-U-per-L

Value set that contains codes for labs that may report their quantitative results in U/L
QuantitativeLaboratoryCodes-fL

Value set that contains codes for labs that may report their quantitative results as fL
QuantitativeLaboratoryCodes-g-per-L

Value set that contains codes for labs that may report their quantitative results as g/L
QuantitativeLaboratoryCodes-g-per-dL

Value set that contains codes for labs that may report their quantitative results as g/dL
QuantitativeLaboratoryCodes-mL-per-min

Value set that contains codes for labs that may report their quantitative results as mL/min
QuantitativeLaboratoryCodes-mg-per-dL

Value set that contains codes for labs that may report their quantitative results as mg/dL
QuantitativeLaboratoryCodes-million-per-uL

Value set that contains codes for labs that may report their quantitative results in 10*6/uL
QuantitativeLaboratoryCodes-mmol-per-L

Value set that contains codes for labs that may report their quantitative results as mmol/L
QuantitativeLaboratoryCodes-percent

Value set that contains codes for labs that may report their quantitative results in %
QuantitativeLaboratoryCodes-pg

Value set that contains codes for labs that may report their quantitative results as pg
QuantitativeLaboratoryCodes-ratio

Value set that contains codes for labs that may report their quantitative results as {ratio}
QuantitativeLaboratoryCodes-thousand-per-uL

Value set that contains codes for labs that may report their quantitative results in 10*3/uL
QuantitativeResultInterpretationVS

Interpretations of abnormal quantitative results.
RadiationProcedureVS

Radiation oncology AND/OR radiotherapy (procedure)
StageGroupVS

Classification of the extent of the disease, according to the classification system’s rules, typically taking into account several anatomic and prognostic elements. This value set intends to cover content contained in LOINC answer lists LL240-3 and LL234-6.
SurgicalProcedureVS

Surgical procedure (procedure)
TNMPrimaryTumorClassificationVS

Possible classifications of the primary tumor according to TNM staging rules.
TNMTumorStagingSystemVS

System used for TNM classification.
TreatmentIntentVS

The purpose of a treatment. Examples include curative, palliative, and supportive.
TreatmentTerminationReasonVS

Values used to describe the reasons for stopping a treatment. Includes code for ‘treatment completed’ as well as codes for unplanned (early) stoppage. Applies to medications and other treatments that take place over a period of time, such as radiation treatments.
TumorMarkerVS

A tumor marker is a substance found in tissue or blood or other body fluids that may be a sign of cancer or certain benign (noncancer) conditions. This value set of LOINC codes will include a comprehensive list of tumor markers across all cancers, and regardless of method or specimen type. Subsets specific to a cancer type will be created as needed. Sources vary but include https://labtestsonline.org/tests/tumor-markers.
