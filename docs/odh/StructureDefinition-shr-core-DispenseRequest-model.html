DispenseRequest

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/odh/StructureDefinition/shr-core-DispenseRequest-model

Indicates the specific details for the dispense or medication supply part of a medication request (also known as a Medication Prescription or Medication Order). Note that this information is not always sent with the order. There may be in some settings (e.g. hospitals) institutional or system support for completing the dispense details in the pharmacy department.

This logical model was published on Wed May 08 00:00:00 EDT 2019 as a draft by HL7.

This IG does not define a corresponding profile for this logical model.
