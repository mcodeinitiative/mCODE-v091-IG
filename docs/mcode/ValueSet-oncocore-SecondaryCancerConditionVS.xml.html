<p><b>SecondaryCancerConditionVS ValueSet</b></p>
<p>Types of secondary malignant neoplastic disease. Includes SNOMED CT disorder hierarchy codes for secondard malignant neoplastic disease, as well as ICD-10-CM codes for malignant neoplasms and neoplasms of uncertain or unspecified behavior. If body site is not precoordinated (implied by the code), it shall be separately specified. For ICD-10-CM coding of the secondary disease, the disorder code should always be SCT#128462008 (Secondary malignant neoplastic disease), and histology/morphology/behavior code and the topology code (body site) and should be specified separately in the HistologyMorphologyBehavior extension and BodySite, respectively.</p>
</div>
