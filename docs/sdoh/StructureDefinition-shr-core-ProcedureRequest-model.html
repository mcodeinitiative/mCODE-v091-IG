ProcedureRequest

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-ProcedureRequest-model

A request for a procedure to be performed. May be a proposal or an order.

This logical model was published on Mon Apr 01 00:00:00 EDT 2019 as a draft by MITRE Corporation.
