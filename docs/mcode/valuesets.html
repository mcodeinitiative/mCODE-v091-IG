BodyLocationVS

Codes that describe normal and pathologic anatomic systems, regions, cavities, and spaces. This value set is part of the profile on body site appearing throughout the mCODE implementation guide. Body location is a flexible structure that allows the location to be determined by a single code, or a code plus laterality and/or orientation. The body location can also be specified in relation to one or more body landmarks. SNOMED CT is used in all cases.

  • Code only: The code should include (precoordinate) laterality and/orientation to the degree necessary to completely specify the body location.
  • Code plus laterality and/or orientation: The basic code augmented by codes specifying the body side and/or anatomical orientation.
  • Relation to landmark: The location relative to a landmark is specified by:
  • Establishing the location and type of landmark using a body site code and optional laterality/orientation, and
  • Specifying the direction and distance from the landmark to the body location.

Note that BodyLocation is a data type (a reusable structure), not a stand-alone entity. The concept is similar to how a postal address can apply to a person, location, or organization. This contrasts with FHIR’s stand-alone BodySite (aka BodyStructure in r4) which ‘is not … intended for describing the type of anatomical location but rather a specific body site on a specific patient’ (FHIR 3.5).

CancerBodyLocationVS

Codes describing the location of primary or secondary cancer. The value set includes all codes SNOMED-CT body structure hierarchy. The cancer body location code can also come from ICD-O-3 topography codes, however, those codes are not included here due to intellectual property restrictions. Because the value set binding is extensible in the information model, use of ICD-O-3 topography codes conform to mCODE requirements.

CancerDisorderVS

A broad cancer-related value set containing both primary and secondary tumor types, with codes from ICD-10 and SNOMED CT, including both diagnosis and histology/morphology/behavior codes. ICD-O-3 codes are not included here due to intellectual property restrictions, but the value set binding is extensible, so ICD-O-3 codes conform to mCODE requirements.
CancerHistologicGradeVS

The classification for the grade of the tumor. This is a subset of the LOINC answer list LL213-0, which represents allowable values for NAACCR data item #440. The original answer list is outdated, as it includes terms pointing to the lineage of hematopoietic/lymphoid neoplasms, which have been retired by NAACCR as of version 18 Data Standards and Data Dictionary.

ConditionCategoryVS

DetectedNotDetectedVS

Value set consisting of the answers Detected (Present) and NotDetected (Absent). The terms detected and not detected are preferred because they reflect outcomes of experimental observation, while present and absent suggest ground truth. This value set reconciles or combines LOINC answer lists LL956-4, LL750-1, LL744-4, LL955-6, LL3950-4, LL3250-9, LL2872-1, LL2410-0, LL2355-7.
GeneticTestMethodVS

The method used to perform the genetic test. Examples include Fluorescent in situ hybridization (FISH), polymerase chain reaction-based assays (PCR), Sanger sequencing, and next-generation sequencing (NGS).

FHIR is currently unable to handle values from NCI Thesaurus, so no values are listed in the Expansion, below. The values ARE in the downloads. GenomicSourceClassVS

The genomic class of the specimen being analyzed: Germline for inherited genome, somatic for cancer genome, and prenatal for fetal genome. (source: LOINC). These are answers to the question specimen source (SCT#703065002).
HistologyMorphologyBehaviorVS

The histologic type codes can be taken either from ICD-O-3 morphology codes (including behavior suffix) or SNOMED CT neoplasm morphology codes. For primary cancers, the ICD-O-3 behavior suffix should be /1, /2, or /3. For secondary cancers, the ICD-O-3 behavior suffix should be /6 or /9. The ICD-O-3 codes are not included here due to intellectual property restrictions, but the value set binding is extensible, so ICD-O-3 codes conform to mCODE requirements.
PrimaryCancerDisorderVS

Types of primary malignant neoplastic disease, coded in SNOMED CT, ICD-10-CM, or ICD-O-3. *SNOMED CT coding: Use a code from the disorder hierarchy under SNOMED 363346000 ‘Malignant neoplastic disease (disorder)’ exluding codes descending from SNOMED CT 128462008 ‘Secondary malignant neoplastic disease (disorder)’. If body site is not precoordinated (i.e., implied by the code), the site of the primary tumor should be specified in the body site attribute.

  • ICD-10-CM coding: Use one of the codes given in this value set representing primary malignant neoplasms and neoplasms of uncertain or unspecified behavior.
  • ICD-O-3 specifies morphology and topography, not disorder; mCODE requires in this case that the disorder code must be SNOMED-CT 363346000 ‘Malignant neoplastic disease (disorder)’. The ICD-O-3 histology/morphology/behavior code and the topography code should then be specified in ICD-O-3 codes in the fields HistologyMorphologyBehavior extension and bodySite, respectively.
QuantitativeResultInterpretationVS

Interpretations of abnormal quantitative results.
RadiationProcedureVS

Codes describing radiation therapy procedures. Includes codes from SNOMED CT and ICD-10-PCS. ICD-10-PCS codes are not constrained. CPT codes are not explicitly included due to licensing restrictions.
SecondaryCancerDisorderVS

Types of secondary malignant neoplastic disease, coded in SNOMED CT, ICD-10-CM, or ICD-O-3.

  • SNOMED CT coding: Use a code from the disorder hierarchy under secondary malignant neoplastic disease (SNOMED-CT 128462008).
  • ICD-10-CM coding: Use one of the codes given in this value set representing secondary malignant neoplasms and neoplasms of uncertain or unspecified behavior. If body site is not precoordinated (implied by the code), it should be specified separately using the body location. *ICD-O-3 specifies morphology and topography, not disorder; mCODE requires in this case that the disorder code must be SNOMED-CT 128462008 (Secondary malignant neoplastic disease). The histology/morphology/behavior code and the topography code (body site) should then be specified in ICD-O-3 codes in the fields HistologyMorphologyBehavior extension and bodySite, respectively.
SurgicalProcedureVS

Codes describing surgical procedures. Includes codes from SNOMED CT and ICD-10-PCS. ICD-10-PCS codes are not restricted due to technical limitations, but should be constrained to codes in the Medical and Surgical section. CPT codes are not explicitly included due to licensing restrictions.
TNMDistantMetastasesClassificationVS

Classifications of distant metastases according to TNM staging rules.
TNMPrimaryTumorClassificationVS

Classifications of the primary tumor according to TNM staging rules.
TNMRegionalNodesClassificationVS

Classifications of regional lymph nodes according to TNM staging rules.
TNMStageGroupVS

Classification of the extent of the disease, according to the TNM staging rules, typically taking into account several anatomic and prognostic elements.
TNMTumorStagingSystemVS

System used for TNM classification. Includes the following codes: SCT#444256004 ‘American Joint Commission on Cancer, Cancer Staging Manual, 6th edition neoplasm staging system (tumor staging)’ SCT#443830009 ‘American Joint Commission on Cancer, Cancer Staging Manual, 7th edition neoplasm staging system (tumor staging’ #AJCC8 ‘American Joint Commission on Cancer, Cancer Staging Manual, 8th edition neoplasm staging system (tumor staging)’ SCT#258235000 ‘International Union Against Cancer (tumor staging)’
TreatmentIntentVS

The purpose of a treatment. A simplified set taken from SNOMED, only including curative and palliative. Curative is defined as any treatment meant to reduce or control a disease process, even if a ‘cure’ is not anticipated. Palliative includes treatments meant to reduce symptoms and side effects, such as antiemetics.
TreatmentTerminationReasonVS

Values used to describe the reasons for stopping a treatment. Includes code for ‘treatment completed’ as well as codes for unplanned (early) stoppage. Applies to medications and other treatments that take place over a period of time, such as radiation treatments.
TumorMarkerTestVS

A tumor marker is a substance found in tissue or blood or other body fluids that may be a sign of cancer or certain benign (noncancer) conditions. This value set of LOINC codes will include a comprehensive list of tumor markers across all cancers, and regardless of method or specimen type. Subsets specific to a cancer type will be created as needed. Sources vary but include https://labtestsonline.org/tests/tumor-markers. At the current time, profiles for the LOINC tests do not exist.
