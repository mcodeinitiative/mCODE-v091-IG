Practitioner

 

The official URL for this profile is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-Practitioner

A person who practices a healing art. HumanName is limited to one required value by US-Core. Gender is required by US-Core.

This profile builds on Practitioner.

This profile was published on Wed Mar 13 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
