ReferralRequest

 

The official URL for this profile is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-ReferralRequest

Used to record and send details about a request for referral service or transfer of a patient to the care of another provider or provider organization.

This profile builds on ReferralRequest.

This profile was published on Wed Mar 27 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
