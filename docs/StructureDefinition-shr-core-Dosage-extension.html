Dosage

 

The official URL for this extension is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-Dosage-extension

The dosage of the medication. The data structure is flexible and can describe a dosage prescribed, administered, or reported.

This extension was published on Wed Mar 13 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
