ActionRequested

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-ActionRequested-model

Abstract class representing a request for any type of action to be performed.

This logical model was published on Sat Mar 30 00:00:00 EDT 2019 as a draft by HL7.
