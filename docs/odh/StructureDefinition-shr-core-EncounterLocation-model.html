EncounterLocation

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-EncounterLocation-model

Locations where the patient has been.

This logical model was published on Sat Mar 30 00:00:00 EDT 2019 as a draft by HL7.

This IG does not define a corresponding profile for this logical model.
