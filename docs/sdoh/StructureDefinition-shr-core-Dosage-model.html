Dosage

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-Dosage-model

The dosage of the medication. The data structure is flexible and can describe a dosage prescribed, administered, or reported.

This logical model was published on Mon Apr 01 00:00:00 EDT 2019 as a draft by MITRE Corporation.

View the corresponding Dosage profile.
