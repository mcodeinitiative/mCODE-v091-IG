Appointment

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/odh/StructureDefinition/shr-core-Appointment-model

A booking of a healthcare event among patient(s), practitioner(s), related person(s) and/or device(s) for a specific date/time.

This logical model was published on Wed May 08 00:00:00 EDT 2019 as a draft by HL7.
