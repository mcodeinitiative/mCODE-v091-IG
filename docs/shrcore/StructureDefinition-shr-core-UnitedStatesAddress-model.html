UnitedStatesAddress

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-UnitedStatesAddress-model

An address for a place in the USA, conforming to US mail postal conventions. (Source: HL7 FHIR).

This logical model was published on Tue May 07 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
