BodyWeight

 

The official URL for this profile is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-BodyWeight

The mass or quantity of heaviness of an individual. This LOINC code should not be used for fetal body weight estimates.

This profile builds on Observation.

This profile was published on Mon Apr 08 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
