EnableWhen

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-EnableWhen-model

A constraint indicating that this item should only be enabled (displayed/allow answers to be captured) when the specified condition is true.

This logical model was published on Tue May 07 00:00:00 EDT 2019 as a draft by The MITRE Corporation.

This IG does not define a corresponding profile for this logical model.
