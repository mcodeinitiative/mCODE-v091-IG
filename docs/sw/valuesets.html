Primary local value sets used in this Implementation Guide

Name Definition
BodyLocationVS

Codes that describe normal and pathologic anatomic systems, regions, cavities, and spaces.
ClockFaceDirectionVS

The relative direction of an object described using the analogy of a 12-hour clock to describe angles and directions. One imagines a clock face lying either upright or flat in front of oneself, and identifies the twelve hour markings with the directions in which they point.
DetectedNotDetectedVS

Value set consisting of the answers Detected (Present) and NotDetected (Absent). The terms detected and not detected are preferred because they reflect outcomes of experimental observation, while present and absent suggest ground truth. This value set reconciles or combines LOINC answer lists LL956-4
MucousMembraneIntegrityVS

Codes describing integrity of mucous membranes.
PeriwoundConditionVS

Provides values for the periwound condition, the state of the skin surrounding a wound.
PodiatryBodyLocationVS

Locations and landmarks relating to the foot.
PressurePointVS

A set of codes that describe locations on the body that are succeptible to pressure ulcers.
RecurrenceVS

A code that describes the number of recurrence of an event.
SkinColorVS

The color or hue of the patient’s skin, compared to its normal color.
SkinDegreeOfMoistureVS

A set of codes that describe how moist the skin is.
SkinIntegrityVS

A set of codes that describe the integrity of the skin.
SkinTemperatureVS

A set of codes that describe the temperature of the skin.
SkinTurgorVS

Values that describe the skin’s ability to return to normal position after being pinched and pulled up.
WoundAssessmentInterpretationVS

Overall assessment of the wound.
WoundBedAppearanceVS

A set of codes that describe how the wound bed looks.
WoundBedEdgeColorVS

A set of codes that describes the color of the wound bed.
WoundDepthQualitativeVS

Codes that describe the depth of a wound in qualitative terms.
WoundEdgeDescriptionVS

A set of codes that describe what the boundaries of the wound look like.
WoundEpisodeVS

A set of codes describing the episodic state.
WoundExudateAmountDescriptionVS

A set of codes that give a qualitative description of the volume of wound exudate.
WoundExudateAppearanceVS

A set of codes that describe what the exudate looks like.
WoundExudateColorVS

A set of codes describing the color of the exudate.
WoundExudateOdorVS

A set of codes that describe how the exudate smells.
WoundGradationVS

A set of codes that describe the stage of the wound or the degree of a burn.
WoundTrendVS

A set of codes that describe the way in which something is developing or changing.
WoundTypeEtiologyVS

A set of codes that describe the causitive agent of a wound.
WoundTypeVS

Codes that describe a kind of wound.
WoundVisibleInternalStructureVS

A set of codes that describe things seen within a wound.
YesNoUnknownVS

A value set containing yes, no, and unknown.
