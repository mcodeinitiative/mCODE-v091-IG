Questionnaire

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-Questionnaire-model

A structured set of questions intended to guide the collection of answers from end-users. Questionnaires provide detailed control over order, presentation, phraseology and grouping to allow coherent, consistent data collection.

This logical model was published on Wed May 01 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
