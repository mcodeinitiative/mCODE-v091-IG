CancerHistologicType

 

The official URL for this profile is: 

http://hl7.org/fhir/us/fhirURL/StructureDefinition/oncocore-CancerHistologicType

Classification of the cancer by the type of tissue in which the cancer originates. (Source: NCI)

This profile builds on http://fhir.org/guides/argonaut/StructureDefinition/argo-observationresults.

This profile was published on Fri Feb 08 00:00:00 EST 2019 as a draft by The MITRE Corporation.
