EncounterDiagnosis

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-EncounterDiagnosis-model

A diagnosis, which is a disease or injury determined to be present through evaluation of patient history, examination, and/or review of laboratory data. This element is not a stand-alone representation of the condition that was diagnosed, but is meant to be used as an sub-structure in a resource, such as Encounter.

This logical model was published on Sat Mar 30 00:00:00 EDT 2019 as a draft by HL7 Patient Care and Clinical Information Modeling Initiative Work Groups.

This IG does not define a corresponding profile for this logical model.
