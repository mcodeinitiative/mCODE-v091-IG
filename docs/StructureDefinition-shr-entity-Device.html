Device

 

The official URL for this profile is: 

http://hl7.org/fhir/us/fhirURL/StructureDefinition/shr-entity-Device

A specific durable physical device used in diagnosis or treatment. The value is the coding for a type of device, for example, a CPAP machine. The same device might be used on multiple patients.

This profile builds on http://fhir.org/guides/argonaut/StructureDefinition/argo-device.

This profile was published on Fri Feb 08 00:00:00 EST 2019 as a draft by The MITRE Corporation.
